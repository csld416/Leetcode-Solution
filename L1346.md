# 1346. Check If N and Its Double Exist

## Solution

Brute force, thatâ€™s all it takes.

## Code

```cpp
class Solution {
   public:
    bool checkIfExist(vector<int>& arr) {
        for (int i = 0; i < arr.size(); i++) {
            for (int j = 0; j < arr.size(); j++) {
                if (i != j && arr[i] == 2 * arr[j]) {
                    return true;
                }
            }
        }
        return false;
    }
};

```
